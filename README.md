# QR Code Genertaor in Python

## 📍Description
This project can generate multiple QR codes dynamically for your every URL by just taking the URL and name of the file from user as an input and saving them without overwriting previous one. 

### 🚀Features
- Generates unlimited QR codes
- User-defined file name
- Automatically creates folder if not present

### ⚙Technologies Used
- Python
- qrcode library
- os module

### ▶How to Run
python Project_QR.py
